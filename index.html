Greetings, folks, and welcome to the new wissmiller.net.

This is a first shot at moving my website, wissmiller.net, to Github Pages. The website was hosted on a Raspberry Pi 4 web server in my house for many years, but we have now switched from Comcast to T-Mobile home internet, and the T-Mobile gateway doesn't allow me to set up port forwarding, etc. to make the website work. However, Github Pages supports free static websites, so this will be the new home of wissmiller.net.

To get on the email list for family news, such as genealogy and the annual Wissmiller reunion, please send me an email. (See Contact Me.)
